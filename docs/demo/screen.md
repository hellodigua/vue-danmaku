# 大屏模式

大屏模式主要用于在大型显示屏幕上展示弹幕，如商场广告屏、会议大屏、展览屏等场景，特点是屏幕较大、弹幕数量多、样式丰富。

## 交互式大屏弹幕

在会场或活动现场，观众可以通过扫码或其他方式发送弹幕，实时显示在大屏上，增强互动体验。
